Handling Text

Formatting Text

The <b> tag makes text bold, the <i> tag makes text italic, and the <u> tag underlines the text. To highlight text, use the <mark> tag, which will result in highlighted text. The <s> tag strikes through the text. Headers are defined using the <h1> to <h6> tags, where

Additionally, the <strong> tag emphasizes text strongly, making it bold, and the <em> tag adds emphasis, resulting in italic text. To create smaller text, use the <small> tag, as seen in this example. Subscript text can be created using the <sub> tag, like in H2O, and superscript text with the <sup> tag, as in E=mc2.
Math Environments and equations

The <slide> tag represents a single slide in a math presentation. Within a slide, the <title> tag is used to define the title of the slide. The main content of the slide is enclosed within the <content> tag.

The <theorem-block> tag is used to represent a theorem. It can have an id attribute for referencing, and a pause attribute that might be used for controlling the presentation flow. Inside the theorem block, the <title> tag specifies the title of the theorem, and the statement follows. The <equation-block> tag is used to include mathematical equations, which are rendered as LaTeX.

Similarly, the <proposition-block> tag is used for propositions. It can also have attributes like id, pause, and class for additional control or styling. The statement of the proposition follows, and the <align-block> tag is used to include aligned equations, often with multiple lines or labels.

Examples of the tags in use 
<slide>
  <title>Math Environments</title>
  <content>
    <theorem-block id="thm" pause>
      <title>Theorem's title</title>
      The statement goes here.
      <equation-block> \int_a^b f(x) {\rm d}x = F(b) - F(a) </equation-block>
    </theorem-block>
    <proposition-block id="prop" pause class="step">
      The statement goes here.
      <align-block> a & = b+c \\ & = d+e\label{eq} </align-block>
    </proposition-block>
  </content>
</slide>

Reference and Citations

 References

In this system, you can reference any element that has an id attribute. This includes not only mathematical environments like theorems and propositions, but also equations, sections, or any other element with an identifier.

To reference an element, use the <a-ref> tag for general elements like propositions or theorems, and the <eq-ref> tag for equations. These tags take a to attribute, which should match the id of the element you want to reference.

Animation Step-wise LaTeX Equation Animation

In this system, LaTeX equations can be animated step by step as the user navigates through the items using the right arrow key. This feature allows users to progressively reveal different parts of the equation, enhancing clarity and helping them follow the derivation.

The syntax works by adding the animate attribute to the equation-block environment and enclosing each part of the equation in curly braces { }. For example, the equation {a} {=} {b{+}c} will be revealed in four steps:

  1. The first step reveals the variable a.
  2. The second step reveals the equals sign =.
  3. The third step reveals the variables b and c.
  4. The final step reveals the addition sign + between b and c.

Citation and Bibliography How to Create a Bibliography Item

To add bibliography items in DELTA, use the <the-bibliography> tag to wrap all references. Each reference is created using the <bib-item> tag. The following fields are required:

  • <authors>: Names of the authors
  • <year>: Year of publication
  • <journal>: Name of the journal or other source
  • <title>: Title of the work

Optionally, you can add the <type> tag for works that are not journal articles (e.g., Arxiv).

Example of a bibliography item:

<bib-item id="osmar">
  <authors>C. Alves and R. Ribeiro</authors>
  <year>2020</year>
  <journal>Journal of Statistical Physics</journal>
  <title>On Mechanical Waves</title>
</bib-item>

You can add as many <bib-item> entries as needed, and they can be placed in any order inside the <the-bibliography> section.

How to Cite Bibliography Items

After creating your bibliography items, you can reference them in your presentation using the <cite-work> tag. This tag accepts a comma-separated list of IDs corresponding to the works you want to cite.

Example of citing works:

<cite-work ids="osmar,paper2,work"></cite-work>

This will create a citation that references the works with the IDs osmar, paper2, and work, in that order. The IDs must match the id attribute of the <bib-item> tags you previously defined. See it in action . Try hovering the cursor over the numbers.

Make sure the id attributes in your bibliography items are unique, as they allow you to reference specific works within the text.
